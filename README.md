# Universelle-Ariana

Une plateforme web inclusive et accessible pour rendre visibles les cas humanitaires urgents à Ariana, Tunisie. Notre mission est de connecter les donateurs aux bénéficiaires de manière transparente et efficace, tout en garantissant l'accessibilité pour tous, y compris les personnes malvoyantes ou en situation de handicap.


## Problème Actuel
Malgré 44 membres actifs et un impact sur +300 000 bénéficiaires, l’association UniVersElle Ariana peine à :
- **Rendre visibles les cas les plus urgents** (enfants démunis, personnes âgées abandonnées, SDF, orphelins).
- **Établir un lien de confiance transparent avec les donateurs**.
- **Centraliser les demandes d’aide dispersées**.

Chaque jour, des vies attendent en silence. Chaque jour, des dons potentiels se perdent dans la méfiance.

## Notre Solution : Miaza
Miaza est une plateforme web conçue pour :
- **Centraliser les demandes d’aide** et les rendre visibles.
- **Créer un lien transparent entre donateurs et bénéficiaires**.
- **Garantir l’accessibilité** pour tous, y compris les personnes malvoyantes ou en situation de handicap.
- **Faciliter la gestion des cas urgents** pour les membres de l’association.

### Fonctionnalités Clés
- **Tableau de bord pour les membres** : Gestion des cas, suivi des dons, et reporting.
- **Interface accessible** : Conforme aux normes WCAG pour l’accessibilité.
- **Système de confiance** : Transparence totale sur l’utilisation des dons.
- **Base de données centralisée** : Pour éviter la dispersion des informations.
